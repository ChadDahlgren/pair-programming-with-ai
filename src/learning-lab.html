Lab 1: Building a Validation Function

▶
Video Demonstration

Scenario

You need to validate user input for a registration form, but the requirements aren't fully defined yet. Through conversation with AI, you'll discover what validation rules are actually needed while building the solution iteratively.

What You'll See

This demonstration walks through the entire process of using AI as a pair programming partner. You'll see how to start with a rough idea, let AI help clarify requirements, handle edge cases through conversation, and arrive at a robust solution through multiple refinement cycles.

Key Takeaways

  • You don't need perfect requirements before starting
  • AI can help you discover what you actually need
  • Iteration is faster than trying to be perfect the first time
  • Context accumulates with each exchange

Lab 2: Refactoring Legacy Code

▶
Video Demonstration

Scenario

You've inherited a messy codebase with technical debt. The code works, but it's hard to maintain and doesn't follow modern patterns. Learn how to use AI to refactor systematically while preserving existing behavior.

What You'll See

Watch how to give AI the right context about existing code, break a large refactor into manageable steps, verify behavior is preserved at each stage, and make informed decisions about when to accept AI suggestions versus when to redirect.

Key Takeaways

  • Refactoring with AI is about strategy, not just automation
  • You stay in control of architectural decisions
  • Testing and verification are part of orchestration
  • Complex work happens through multiple small steps
