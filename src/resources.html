Resources & Tools

AI tools and resources I use and recommend for effective pair programming. Each has unique strengths for different aspects of the orchestration model.

Recommended AI Tools

Cursor

My primary AI coding environment. Cursor excels at understanding your codebase context and maintaining continuity across conversations. Best for iterative development, refactoring, and working within existing projects.

Visit Cursor

TalkTastic

AI-powered voice dictation for macOS that transforms speech into polished text. Uses context-aware AI to adapt your words to fit the tone and style of any application. Perfect for dictating code comments, documentation, and rapid text input across your workflow.

Visit TalkTastic

Claude Code

An AI-powered coding assistant that integrates directly into your development environment. Offers intelligent code completion, real-time collaboration, and full-stack development capabilities powered by Anthropic's Claude models.

Visit Claude Code
